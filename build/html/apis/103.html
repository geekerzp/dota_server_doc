103角色登陆游戏场景¶

request

1
2
3
4
5
 {
     "userId": "xxx",
     "characterId": "xxx",
     "sceneType": "dota"
 }
  1. “userId”是用户id。
  2. “characterId”是用户角色id。
  3. “sceneType”是游戏场景类型，可选择为”pvp”或者为”dota”。

     "message": "xxx",
     "data": {
         "name": "xxx",
         "cid": 1,
         "level": 1,
         "maxexp": 1,
         "exp": 1,
         "coin": 1,
         "yuanbao": 1
     }
 }
  1. “result”是用户id。
  2. “message”是请求提示信息。
  3. “name”是角色昵称。
  4. “cid”是角色id。
  5. “level”是角色等级。
  6. “maxexp”是角色最大经验值。
  7. “exp”是角色经验值。
  8. “coin”是金币。
  9. “yuanbao”是元宝。
