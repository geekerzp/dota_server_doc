9001移动事件
============

*request*

.. code-block:: json
   :linenos:

    {
        'characterId': xxx,
        'roomId': xxx,
        'data': {
            'timestamp': xxx,
            'actorId': xxx,
            'destination': {
                'x1': xxx,
                'y1': xxx,
                'x2': xxx,
                'y2': xxx
            }
        }
    }

#. 'characterId'角色id。
#. 'roomId'对战房间id。
#. 'timestamp'时间戳，和服务器对时后的虚拟时间。
#. 'actorId'英雄的战场id。
#. 'destination'目的坐标。
