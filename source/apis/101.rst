101用户登陆服务器
=================

*request*

.. code-block:: json
   :linenos:


   {
       'username': 'xxx',
       'password': 'xxx'
   }

#. 'username'是游戏用户用户名。
#. 'password'是游戏用户密码。

*response*

.. code-block:: json
   :linenos:


   {
     'result': True or False,
     'data': {
         'userId': 'xxx',
         'hasRole': True or False,
         'characterId': 'xxx'
     }
   }

#. 'result'是请求执行结果。
#. 'userId'是用户id。
#. 'hasRole'是用户是否已经创建游戏角色。
#. 'characterId'是用户游戏角色id。
